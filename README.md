## Abstract

We show that the coordinate speed of light in a gravitational field,
c_eff/c = exp(Phi/c^2), equals the lower bound on the Petz recovery
fidelity: c_eff/c = exp(-Sigma_grav/2). The gravitational entropy production
Sigma_grav = 2 ln Q = -ln(-g_00) is exact for stationary spacetimes,
where Q = 1/sqrt(-g_00) is the inverse Khronon lapse. This identifies
gravitational decoherence as a Petz recovery problem and the event
horizon as the point of complete information loss (tau = 1).
